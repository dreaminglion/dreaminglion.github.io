起泡排序

把一个数组的元素，安装从小到大的顺序排列。

解析：每执行一轮，得到一个数组最大的元素。数组剩下需要排序的元素为n-1。而且7个元素，只需要比较6次就可以得到最大值，所以流程如下：
for (int j = 0; j < n - 1 - 0; j++)
for (int j = 0; j < n - 1 - 1; j++)
for (int j = 0; j < n - 1 - 2; j++)
for (int j = 0; j < n - 1 - n; j++)

所以得到排序结果需要比较
